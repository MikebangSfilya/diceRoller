# diceRoller

## Что за штука?

Небольшое мини приложение для автоматического расчета инициативы в условиях боевой системы WoD:  [подробнее](https://whitewolf.fandom.com/wiki/Combat_Systems#Stage_One:_Initiative).

Пользователь вводит имена и модификаторы(ловкость/Dexterity, мудрость/Wits) любым продуманным способом(на данный момент только JSON через http-запрос), в будущем планируется принимать csv файлы, пользовательский ввод, и чтение файлов.
